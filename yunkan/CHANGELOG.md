# Changelog

## 0.9.16

镜像 tag 跟随主项目升级 0.9.15 → 0.9.16。加载项选项、端口、卷挂载**均无变化**。

本次核心是 AI 检测实时性根治：

- **修复多台摄像头同时 AI 检测时检测明显「慢半拍」**：摄像头较多时，AI 检测可能落后实际画面几十秒 —— 事件中心里事件要等约 30 秒才冒出来，从事件点「跳转回放」也会停在事件之后、错过关键画面。现已根治，检测恢复实时：事件几乎立即出现，跳转回放也能精准停在事件发生那一刻。摄像头越多，改善越明显。
- **进一步修复部分 H.265（HEVC）相机的黑屏 / 无声**：少数 H.265 相机在特定情况下直播黑屏或听不到声音，现已修复。
- **对讲更稳**：修复对讲偶发的连接失败。
- **网页端云台操作升级**：网页直播支持群晖 NVR 风格的云台操作（滚轮变焦、按住方向键转动云台），设备管理直播弹窗支持主 / 子码流切换。
- **手机端回放支持横屏全屏 + 双指缩放 / 拖动；登录支持只填公网地址即可**（安卓 / iOS）。
- **系统更新显示下载进度**：在线更新时新增实时下载进度条，下载到哪一步、下了多少一目了然，不用再盯着空白干等。
- **其它优化**：移除手机端一个无效的运动检测开关；缓解偶发的数据库占用导致的卡顿。

## 0.9.15

镜像 tag 跟随主项目升级 0.9.14 → 0.9.15。加载项选项、端口、卷挂载**均无变化**。

本次为修复 + 多语言完善：

- **修复「忽略区域」对鸟等目标不生效**：把检测区域设为「忽略区」并选定目标（例如鸟）、框住整个画面后，画面里的鸟仍反复触发告警 —— 现已修复。忽略区现在会按你选的目标类型一律忽略，不再因为目标置信度高低而漏掉，飞鸟 / 晃动树影等误报可干净屏蔽。
- **英文 / 日文 / 繁体界面补全多处遗漏翻译**：检测区域设置、通用弹窗、摄像头音频能力、自动化运行记录、仪表盘 AI 检测面板等处此前在非中文界面下仍显示中文，现已全部翻译。
- **修复编辑「国标（GB28181）摄像头」时保存失败**：通过国标协议接入的摄像头（部分 NVR / 行业相机）编辑后保存可能报错，现已修复。

## 0.9.14

镜像 tag 跟随主项目升级 0.9.13 → 0.9.14。加载项选项、端口、卷挂载**均无变化**。

本次为稳定性修复：

- **修复摄像头 AI 检测偶尔莫名停在「待启用」、需手动重开才恢复**：个别普通摄像头在画面正常、无报错时 AI 检测会悄悄卡住，现已修复并新增多重自愈，取流一旦卡住会在十几秒内自动重连恢复，无需手动操作。
- **新增相机「画面冻结（假死）」自动发现与恢复**：摄像头画面卡在某一帧不动、但看起来还在线时，云瞰会识别这种假死并先自动重连尝试唤醒；仍无法恢复则标记异常并提醒，不再默默漏录漏检。
- **修复部分已授权、联网正常的设备 AI 检测打不开**：个别机器授权有效、网络正常、面板显示「已授权」，AI 检测却起不来，现已修复。

## 0.9.13

镜像 tag 跟随主项目升级 0.9.12 → 0.9.13。加载项选项、端口、卷挂载**均无变化**。

本次为功能更新，主要新增 / 改进：

- **接入 Home Assistant 更省心，每台摄像头自动出现**:在设置里填好 MQTT 连接后,家里每台摄像头会自动出现在 Home Assistant 中,无需逐个手动配置——既有摄像头「在线 / 离线」状态,也有检测到「有人 / 车辆 / 动物 / 快递 / 人脸 / 跌倒 / 婴儿哭声」时的实时开关,可直接拿去编排自动化(有人进门 → 开灯、识别到家人 → 撤防、听到婴儿哭声 → 手机提醒)。摄像头增删后自动同步,不残留无效设备。
- **支持国标(GB/T 28181)摄像头与 NVR 一键接入**:海康 / 大华 / 宇视等支持「国标上级平台」的摄像头、NVR、硬盘录像机,只需在设备里填入云瞰的接入信息即自动注册,勾选通道即可添加,一台 NVR 的多通道也能批量导入。接入后直播、录像、AI 检测、快照、云台与预置位、语音对讲、设备端历史回放都与普通摄像头一样使用。
- **延时录像**:可把一段时间的录像生成加速的延时短片,几秒快速回顾一整段。
- **安卓下拉刷新**:监控首页、事件、智能联动页面支持下拉刷新。
- **手机在家 / 在外切换网络更顺、打开更快**:在家连 Wi-Fi、出门用流量时更快更准连上服务器;杀掉重进、或刚切换网络后各页面会自动加载,不用再手动来回切页签;偶尔「明明能用却提示连不上」也一并修复。
- 其它优化与修复:完善语音对讲相关提示文案;修复智能联动表单里人脸名称读取异常;某项后台功能启动失败时如实显示「降级」而非误报「运行中」;本版本同时包含此前测试版对部分 H.265 摄像头「无声」「AI 检测不工作」的修复。

## 0.9.12

镜像 tag 跟随主项目升级 0.9.11 → 0.9.12。加载项选项、端口、卷挂载**均无变化**。

本次为功能更新，主要新增 / 改进：

- **人脸库升级为「人物库」**:可把同一人的多张人脸归到一个人物名下,识别更准更稳;有人由远及近走近时持续收集更清晰的样本,确认是陌生人后才单独提醒,减少误报。已提升为顶层菜单,支持命名 / 合并 / 拆分 / 忽略 / 隐藏,三端同步。
- **更多相机回放 / 直播有声**:采用老式音频编码的摄像头过去在回放、网页、直播里没声音,现已自动转换,回放 / 直播 / 录像均可发声。
- **录像空间管理增强**:可为每台摄像头单独设最大占用 / 保留天数,超限自动清理最旧;115 网盘清理会一并删除云端文件;新增「失效录像」对账,文件已不存在的录像自动隐藏并可一键扫描清理。存储设置并入录像页。
- **网页回放更顺滑**:分段无缝衔接,快退 / 快进 15 秒 + 加载提示,时间轴拖动跟手、滚轮以光标为锚点缩放。
- **离线相机状态更准**:总览 / 实时近实时反映上下线,离线相机改为清晰「离线」占位,不再裂图或一直「连接中」。
- **掉线通知更稳**:画面抖动的相机不再被误判频繁上下线而疯狂推送。
- **服务运行状态一目了然**:新增顶栏用户菜单与服务状态监控。
- **iOS 直播兼容性提升**:修复部分 H.265(HEVC)相机在 iOS 上无画面;直播 / 对讲不再无故占用麦克风。
- **点击通知打开更顺**:先自动判断内外网再加载详情,全程有加载反馈。
- 其它修复:115 网盘按时间段导出整段失败、哭声 / 音频检测稳定性、激活 / 迁移失败提示更清晰(四语言)。
